# ex_pdf_core v1.0.1 - API Reference

## Modules

- [Pdf](Pdf.md): The missing PDF library for Elixir.
- [Pdf.Color](Pdf.Color.md): A list of predefined colors you can use

- [Pdf.Component.Box](Pdf.Component.Box.md): Box component for PDF documents.

- [Pdf.Layout](Pdf.Layout.md): Layout helpers for positioning content in PDF documents.
- [Pdf.Style](Pdf.Style.md): CSS-like styling system for PDF elements.
