# ex_barcode v0.1.0 - API Reference

## Modules

- [ExBarcode](ExBarcode.md): Pure Elixir barcode encoding library.
- [ExBarcode.Code128](ExBarcode.Code128.md): Code 128 barcode encoder.
- [ExBarcode.Shape](ExBarcode.Shape.md): Shape barcode engine — transforms a standard Code 128 barcode into a
shaped barcode with two rendering styles
- [ExBarcode.Shape.Result](ExBarcode.Shape.Result.md): Shaped barcode result — bars + shape data in normalized coords.
